# Zig Systems Engineering Lab

Internal repository for tracking Zig research, systems patterns, and mechanical sympathy studies.

## Project Structure
* `src/memory/`: Allocator studies
* `src/comptime/`: Metaprogramming research
* `src/ffi/`: C Interop patterns
* `src/systems/`: Hardware/OS primitives

## Usage
```bash
zig build run
zig build test
```

For the educational philosophy and curriculum notes, see [docs/philosophy.md](docs/philosophy.md).
